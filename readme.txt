Install plugin to remove related products from your product page

== Description ==

If you Don't want to show Related Products in your WooCommerce product page ? Remove them with this plugin.

Easy to use, you only install and activate the plugin.

After Installing this plugin, Related Products from your product page will be removed. 

== Installation ==

This section describes how to install the plugin and get it working.

== Installation ==

* Download the plugin
* Upload the folder "Hide-Related-Product" to wp-content/plugins (or upload a zip through the WordPress admin)
